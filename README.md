\# StormHaven



A modern weather application with liquid glass design and bilingual support (Greek/English).



\## About



StormHaven provides real-time weather information with an elegant, iOS-inspired interface featuring liquid metal animations and liquid glass design elements.



\## Features



\- \*\*Bilingual Interface\*\* - Full support for Greek and English languages

\- \*\*7-Day Forecast\*\* - Detailed weather predictions with interactive charts

\- \*\*Location Management\*\* - Save and organize multiple weather locations

\- \*\*Responsive Design\*\* - Optimized for desktop, tablet, and mobile devices

\- \*\*Real-time Updates\*\* - Live weather data with smooth animations

\- \*\*Dark Theme\*\* - Modern dark mode interface with liquid glass effects



\## Technologies



\- HTML5

\- CSS3 (Liquid glass, animations)

\- Vanilla JavaScript

\- Anime.js (Animations)

\- ECharts (Data visualization)

\- p5.js (Particle effects)

\- Sortable.js (Drag-and-drop)



\## Pages



\- \*\*Dashboard\*\* (`index.html`) - Current weather and 24-hour forecast

\- \*\*7-Day Forecast\*\* (`forecast.html`) - Weekly weather predictions with charts

\- \*\*Locations\*\* (`locations.html`) - Manage saved locations with drag-to-reorder
